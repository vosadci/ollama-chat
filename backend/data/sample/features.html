Nexus Features

Nexus combines task management, docs, and reporting into a single workspace so your team can focus on work, not switching tools.

Boards

Visualise work in a Kanban-style board. Drag tasks between columns, set WIP limits, and use swimlanes to group by assignee or priority. Boards update in real time for all team members.

Timeline (Gantt chart)

The timeline view shows task dependencies and deadlines on a calendar. Drag the start and end dates of any task directly on the chart. Milestones are displayed as diamonds. Critical path is highlighted automatically.

Docs

Write meeting notes, specs, and wikis inside Nexus using the built-in editor. Docs support markdown, images, tables, code blocks, and @mentions. Any doc can be linked to a project or task.

Time tracking

Log time directly on tasks with the built-in timer or by entering hours manually. Time reports can be exported to CSV or shared with clients. Integrations with Harvest and Toggl are also available.

Automations

Automate repetitive work with no-code rules. Example: "When a task is moved to Done, notify the assignee and close linked GitHub issues." Automations run up to 1,000 times per month on the Pro plan and unlimited on Enterprise.

Reporting

Track velocity, sprint burndown, and team workload from the Reports section. Dashboards are customisable and can be shared via a public link. Data can be exported to CSV or connected to BI tools via the API.

Mobile app

The Nexus iOS and Android apps offer full task management, notifications, and offline support. Changes sync automatically when you reconnect. Available on the App Store and Google Play.
